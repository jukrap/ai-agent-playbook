# Browser DOM Change

Use this as the primary frontend skill for DOM-first and legacy browser behavior.

## Workflow

1. Trace selectors, event delegation, plugin initialization, form serialization, AJAX calls, and server-rendered markup together.
2. Preserve IDs, classes, data attributes, event order, and plugin lifecycle unless the change requires them.
3. Check browser compatibility and script load order before replacing behavior.
4. Verify the actual browser workflow, including initial load, repeated actions, validation failure, and dynamic DOM updates.
