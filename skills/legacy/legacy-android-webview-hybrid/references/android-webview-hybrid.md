# Android WebView Hybrid Stack Checks

- Trace Activity/Fragment lifecycle, WebView settings, asset source, URL loading, JavaScript bridge, native permissions, and web routes together.
- Preserve bridge method names, payload shape, callback protocol, origin checks, user agent behavior, and navigation contracts.
- Check file chooser, downloads, camera/storage permissions, cache/version behavior, offline/error handling, and packaging.
- Verify on emulator or device when native and web behavior cross the bridge.
