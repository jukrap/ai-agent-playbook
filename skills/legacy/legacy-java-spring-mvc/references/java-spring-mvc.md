# Java Spring MVC Stack Checks

- Trace controller mappings, interceptors, filters, JSP fragments, form objects, validators, and model attribute names together.
- Check XML and annotation configuration, bean names, transaction boundaries, MyBatis mapper files, result maps, and stored procedure calls.
- Preserve WAR/container assumptions, servlet filters, session keys, and deployed server configuration.
- Verify GET, POST, validation failure, authorization/session, mapper SQL, and deployment-shaped startup when relevant.
