# jQuery Browser Stack Checks

- Trace active HTML/templates, script include order, selectors, delegated events, plugin initialization, AJAX calls, and globals.
- Preserve IDs, classes, data attributes, plugin lifecycle, CSS hooks, and server-rendered values embedded in JavaScript.
- Watch for duplicate handlers, re-initialization bugs, old syntax constraints, and hidden table/modal/date picker coupling.
- Verify load, changed interactions, repeated actions, console, network, validation failure, and dynamic DOM updates.
