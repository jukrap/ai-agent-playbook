## Rules

- Keep only questions that still affect decisions.
- Close questions with the answer, evidence, and where the answer was promoted.
- Move stale or historical discussion to `archive/`.
- Do not use this file for general notes, task lists, guesses, or worklog history.
- If a question blocks implementation, link it from the active plan or `START_HERE.md`.
