# Browser DOM Change

DOM-first 및 legacy browser behavior를 위한 primary frontend skill입니다.

## Workflow

1. selector, event delegation, plugin initialization, form serialization, AJAX call, server-rendered markup을 함께 추적합니다.
2. 변경이 요구하지 않는 한 ID, class, data attribute, event order, plugin lifecycle을 보존합니다.
3. 동작을 교체하기 전에 browser compatibility와 script load order를 확인합니다.
4. initial load, repeated action, validation failure, dynamic DOM update를 포함해 실제 browser workflow를 검증합니다.
