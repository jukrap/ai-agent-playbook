# UI Polish

보이는 UI 품질을 위한 primary frontend skill입니다.

## Workflow

1. product workflow, existing design system, nearby components, target viewport를 확인합니다.
2. 실제 task 주변의 hierarchy, spacing, layout stability, text fit, accessibility, state coverage를 개선합니다.
3. 사용자가 새 visual direction을 요청하지 않았다면 unrelated redesign은 피합니다.
4. 가능하면 관련 desktop/mobile width에서 rendered UI를 검증합니다.

## Compatibility

`frontend-ui-polish`는 이 skill의 compatibility trigger입니다.
