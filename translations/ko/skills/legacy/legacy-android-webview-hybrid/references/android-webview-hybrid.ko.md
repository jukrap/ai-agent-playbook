# Android WebView Hybrid Stack Checks

- Activity/Fragment lifecycle, WebView setting, asset source, URL loading, JavaScript bridge, native permission, web route를 함께 추적합니다.
- bridge method name, payload shape, callback protocol, origin check, user agent behavior, navigation contract를 보존합니다.
- file chooser, download, camera/storage permission, cache/version behavior, offline/error handling, packaging을 확인합니다.
- native와 web behavior가 bridge를 넘나들면 emulator 또는 device에서 검증합니다.
