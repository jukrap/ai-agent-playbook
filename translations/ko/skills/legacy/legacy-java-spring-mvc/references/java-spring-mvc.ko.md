# Java Spring MVC Stack Checks

- controller mapping, interceptor, filter, JSP fragment, form object, validator, model attribute name을 함께 추적합니다.
- XML/annotation configuration, bean name, transaction boundary, MyBatis mapper file, result map, stored procedure call을 확인합니다.
- WAR/container assumption, servlet filter, session key, deployed server configuration을 보존합니다.
- 관련되면 GET, POST, validation failure, authorization/session, mapper SQL, deployment-shaped startup을 검증합니다.
