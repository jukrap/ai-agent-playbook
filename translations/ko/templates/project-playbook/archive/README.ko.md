# Archive

stale plans, old handoffs, retired prompts, historical notes를 이 폴더에 둡니다.

Archived material은 context이지 active instruction이 아닙니다.

## Rules

- item을 archive한 이유를 짧게 남깁니다.
- 프로젝트가 명시적으로 허용하지 않는 한 private 또는 sensitive archived material을 committed repository에 넣지 않습니다.
- 같은 실수를 반복하지 않게 할 때만 current docs에서 archived material을 링크합니다.
