# Knowledge

외부 source, 채택한 reference, research note를 이곳에 둡니다.

Source owner, status, freshness, locator, credential boundary, promotion policy는 `sources.json`에 기록합니다. 프로젝트에 유용하게 정제한 요약은 `references/`에 두고, 탐색 중인 메모는 `research/`에 둡니다. 외부 reference pattern을 채택, 보류, 거절할지 판단할 때는 `reference-adoption-ledger.md`를 사용합니다.
