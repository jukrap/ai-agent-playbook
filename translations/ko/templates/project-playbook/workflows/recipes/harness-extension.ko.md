# Harness Extension

Inputs: capability gap, target category, proposed skill or tool, permission tier.

Outputs: taxonomy entry, skill 또는 MCP change, docs, tests, translation update.

Skills: agent skill authoring, MCP server design, reference adoption.

Tools: `catalog check`, `workflow list`, skill validation, translation validation.

Stop conditions: duplicate skill name, unclear trigger, unsafe write tier, missing translation plan.

Verification: syntax check, tests, skill validation, translation validation, sync dry run.
